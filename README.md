## URL 版本规则

所有具体版本都必须把版本号放在 URL 后缀路径里区分：

- v1 使用 `/v1/`
- v2 使用 `/v2/`
- v3 使用 `/v3/`
- v4 使用 `/v4/`
- 后续依次使用 `/v5/`、`/v6/` ……

根路径 `/` 只作为版本入口页，不作为具体题库版本。后续新增版本时，只创建新的版本目录，不覆盖旧版本目录。
